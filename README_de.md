# 🗂️ PCMigrator

**PCMigrator** ist ein modulares PowerShell-Skript zur Sicherung und Wiederherstellung von System- und Browserkonfigurationen.  
Es unterstützt Lesezeichen, Erweiterungen und Passwörter der gängigsten Browser, WLAN-Profile, Umgebungsvariablen, installierte Programme und die Hosts-Datei.

![PowerShell](https://img.shields.io/badge/Built_with-PowerShell-5391FE?logo=powershell&logoColor=white)
![License](https://img.shields.io/github/license/IHR-BENUTZERNAME/PCMigrator)

---

## 🌍 Unterstützte Sprachen

- 🇮🇹 Italienisch (Standard)
- 🇬🇧 Englisch
- 🇫🇷 Französisch
- 🇩🇪 Deutsch

Beim Start fragt das Skript nach der gewünschten Sprache und passt alle Menüs und Meldungen entsprechend an.

---

## 📦 Funktionen

- 🔄 Selektive oder vollständige Sicherung/Wiederherstellung
- 🌐 Unterstützung für: Chrome, Firefox, Edge, Opera, Brave
- 🔐 Erweiterungen und Passwörter (je nach Browser verschlüsselt)
- 🖥️ Export von Netzwerkkonfigurationen, WLAN-Profilen, Umgebungsvariablen, Programmen, Hosts-Datei
- 🚦 Interaktive Menüs mit automatischer Rückkehr
- 🧠 Intelligente Erkennung installierter Browser beim Wiederherstellen
